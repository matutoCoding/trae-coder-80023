## 1. 产品概述
快递柜投放管理H5系统，用于管理网点快递柜格口的共享投放、并发扣减、阶梯计费与账单生成。
- 目标用户：快递员、网点管理员
- 解决问题：多快递员共享格口资源时的超额投放、超时滞留计费不透明、账单统计混乱等痛点
- 核心价值：格口资源高效利用、并发安全保障、计费透明可追溯

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 快递员 | 工号注册 | 查看格口余量、投放快递、核销取件码、查看个人账单 |
| 网点管理员 | 管理员账号 | 维护格口配置、设置阶梯费率、查看全网点账单、核销管理 |

### 2.2 功能模块
1. **格口共享模块**：共享额度池、格口大小匹配、实时余量展示
2. **并发扣减模块**：投放加锁、超额拦截、余额一致性校验
3. **阶梯计费模块**：阶梯档位维护、超时滞留计价、用量逐档累加
4. **账单生成模块**：投放明细、费用汇总、取件码核销记录

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 首页Dashboard | 格口概览 | 大/中/小格口余量、今日投放数、滞留件数、待收费用统计卡片 |
| 首页Dashboard | 操作快捷入口 | 一键投放、核销取件、计费设置、账单查询入口 |
| 投放管理页 | 格口选择 | 按快递尺寸自动匹配格口规格、显示可用数量、选择格口 |
| 投放管理页 | 投放表单 | 快递单号录入、收件人手机、预计存放时长、确认投放（含并发锁机制） |
| 投放管理页 | 在途列表 | 显示当前用户投放中快递、存放时长、取件码、状态 |
| 计费设置页 | 阶梯档位表 | 各格口规格的阶梯费率配置（按天递增单价）、新增/编辑/删除档位 |
| 计费设置页 | 计费预览器 | 输入格口类型和存放天数，实时计算阶梯费用展示 |
| 账单查询页 | 账单汇总 | 按日期/快递员筛选、总投放数、总费用、已核销/未核销统计 |
| 账单查询页 | 投放明细 | 单号、格口类型、投放时间、取件时间、存放天数、阶梯费用明细 |
| 取件核销页 | 核销表单 | 输入取件码/扫描二维码、显示快递信息、确认核销结算 |

## 3. 核心流程
用户（快递员）登录系统 → 查看格口实时余量 → 选择格口类型并录入快递信息 → 系统执行并发扣减加锁 → 扣减成功生成取件码 → 快递超时按阶梯费率逐档计费 → 收件人取件或快递员核销 → 生成账单记录

```mermaid
flowchart LR
    A["快递员登录"] --> B["查看格口余量"]
    B --> C["选择格口+录入信息"]
    C --> D{"并发加锁扣减"}
    D -->|成功| E["生成取件码"]
    D -->|失败-余量不足| F["提示重试"]
    E --> G["超时滞留阶梯计费"]
    G --> H["取件/核销"]
    H --> I["生成账单记录"]
```

## 4. 用户界面设计
### 4.1 设计风格
- 主色调：工业蓝 `#1E40AF`，辅助色：警示橙 `#F59E0B`，成功绿 `#10B981`，错误红 `#EF4444`
- 背景色：深灰 `#0F172A`（暗色工业风，适合物流仓储场景）
- 按钮风格：圆角8px，带微投影，hover上浮效果
- 字体：标题用 `Space Grotesk` 现代无衬线，正文用系统字体
- 布局风格：移动端优先卡片式布局，顶部Tab导航，滑动操作区
- 图标风格：Lucide线性图标，统一24px尺寸

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| Dashboard | 格口概览 | 渐变统计卡片、环形进度条、数字滚动动画、实时状态指示灯 |
| 投放管理 | 格口选择 | 尺寸标签切换、格口数量胶囊、匹配高亮、投放表单模态框 |
| 计费设置 | 阶梯档位表 | 阶梯色块、可编辑行、拖拽排序、费用曲线预览图 |
| 账单查询 | 投放明细 | 时间线布局、费用分段高亮、筛选器面板、汇总固定底栏 |
| 取件核销 | 核销表单 | 大号取件码输入框、扫码按钮、核销成功动效 |

### 4.3 响应式
- 移动端优先设计（H5），最大宽度480px居中展示
- 触控区域≥44px，滑动手势支持
- 适配iOS/Android主流浏览器
